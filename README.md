# Installazione
### Lanciare da terminale:
    composer install
    npm install
    duplicare .env.example e rinominarlo .env (modificando i dati necessari)
    php artisan key:generate


### Se si crea un nuovo db (PHPMyAdmin), lanciare da terminale
    php artisan migrate
    php artisan db:seed
